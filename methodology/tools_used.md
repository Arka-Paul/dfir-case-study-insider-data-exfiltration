Tools Used
FTK Imager

Used for forensic acquisition of the virtual machine disk in E01 format and for cryptographic hash generation and verification.

Autopsy

Primary forensic analysis platform used for timeline reconstruction, artefact identification, and correlation across file system and operating system layers.

HxD Hex Editor

Used to inspect file headers and detect manipulation or obfuscation techniques.

CyberChef

Used to decode encoded artefacts and support analysis of concealed credentials and data representations.

VeraCrypt

Analysed encrypted containers to assess secure data storage behaviour and access control.

Wireshark

Reviewed network-related artefacts to support contextual understanding of system activity.
